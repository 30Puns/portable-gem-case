# Portable Gem Case

NeoForge 1.21.1 addon for [Apotheosis](https://www.curseforge.com/minecraft/mc-mods/apotheosis).

## Features

- **Right-click** (in air or on a block) while holding a **Gem Case** or **Ender Gem Case** (House of Fabergé) to open the gem storage GUI from anywhere.
- **Shift + right-click** to place the block normally.
- Gems are stored on the item itself (`BLOCK_ENTITY_DATA`), so they travel with you in your inventory.

## Requirements

- Minecraft 1.21.1
- NeoForge 21.1.x
- Apotheosis 8.5.x

## Development

```bash
gradlew runClient
```

Built JAR: `build/libs/gemcasemod-1.0.0.jar`
